# 🎯 Desafios

Exercícios práticos gerados e corrigidos com o comando `/atividade`, organizados por assunto conforme o avanço nos estudos de Java. Só entram aqui exercícios que valeram a pena guardar como portfólio — nem todo treino rápido é salvo.

## 📁 Estrutura

Cada pasta representa um assunto (ex: `Heranca/`, `Encapsulamento/`, `POO/`) e contém arquivos no padrão `DesafioNN_Nivel.java`, onde `Nivel` é `Facil`, `Medio`, `Dificil` ou `DesafioFinal`. O enunciado de cada exercício está no comentário no topo do próprio arquivo.
