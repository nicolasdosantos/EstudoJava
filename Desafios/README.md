# Desafios

Exercícios práticos gerados e corrigidos com o comando `/atividade`, organizados por assunto conforme avanço nos estudos de Java.

Cada pasta representa um assunto (ex: `Heranca/`, `Encapsulamento/`, `POO/`) e contém arquivos no padrão `DesafioNN_Nivel.java`, onde `Nivel` é `Facil`, `Medio`, `Dificil` ou `DesafioFinal`.

Só entram aqui exercícios que valeram a pena guardar como portfólio — nem todo treino rápido é salvo.
